# aberc

Implementation of aber programming language (https://github.com/Hedgehogo/aber.io) in C.

## Building

Use GCC. Works on windows (maybe) and linux.
```console
$ make
$ ./aberc --help
...or...
$ gcc -o aberc aber.c
```
